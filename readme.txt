Remove the logo of your WordPress control panel with just one click.

== Description ==

With one click removes the logo of your WordPress control panel.

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

* 1 - Baixe o plugin ou instale diretamente do diretorio do WP
* 2 - Clique em ativar
* 3 - Pronto! o logo do WP foi removido com sucesso!
